## 512.1. Modeling Goal

A schema file does not have to define every idea alone. `use` lets one idea file build on shared declarations so projects can start from Stackpress built-ins instead of repeating them.

## 512.2. Use Another Model

Start a Stackpress schema with:

```idea
use "stackpress/stackpress.idea"

model Article {
  id    String @id @default("cuid()")
  title String @is.required("Title is required")
}
```

This imports the Stackpress idea definitions used by the schema processing layer. That is why this detail appears in the lesson before reference material.
