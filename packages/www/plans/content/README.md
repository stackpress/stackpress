# Course Content Drafts

This folder contains course-organized source content for the Stackpress website.
It is intentionally separate from `content-strategy.md`: the strategy explains
the system, while these files contain copy that can be adapted into actual site
pages.

Each file covers one course level:

- `000-orientation.md`
- `100-develop.md`
- `200-data.md`
- `300-build-and-deploy.md`
- `400-project-structure.md`
- `500-idea.md`
- `600-built-ins.md`
- `700-studio.md`
- `800-ai.md`

## Editorial Rules

- Teach Stackpress-facing workflows, not the lower-level sibling libraries as
  separate products.
- Keep course content task-oriented.
- Move exact exports, prop tables, method lists, CLI flags, complete config
  shapes, and exhaustive attribute lists to reference pages.
- Keep all course pages linkable by URL.
- Let the visible navigation expand with reader progress.

## Page Shape

Each course page should be converted into this website shape:

```text
Start here
Quick start
What just happened
Core concepts
Common tasks
Checkpoint
Related reference
Next course
```
