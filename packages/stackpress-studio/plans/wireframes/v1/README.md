# Stackpress Studio Wireframes V1

Grayscale SPA wireframe artifact for Stackpress Studio.

Screens:

 - `index.html` - model workbench
 - `fieldsets.html` - fieldset/type workbench
 - `enums.html` - enum workbench
 - `source.html` - source editor state
 - `references.html` - Cradle reference board

The wireframes intentionally avoid brand color and polished visual styling. They focus on structure, density, navigation, and how Cradle's popup-heavy schema builder maps into a persistent SPA shell.
