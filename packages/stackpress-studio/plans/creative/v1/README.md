Clickable creative design draft for Stackpress Studio.

Cradle functional reference.

## What Is Clickable

- Idea file selection.
- Models, Fieldsets, Enums, and Source navigation.
- Add Model, Add Fieldset, Add Enum, and Add File.
- Model rows, model tabs, nested field editor, icon picker, relation editor,
  copy/remove screens, and drawer back/close.
- Enum option editing and source-file editing.
- Direct review URLs for deeper drawer states:
  `add-model.html`, `add-file.html`, `relation-editor.html`,
  `source-diagnostics.html`, `fieldset-source.html`, and `enum-source.html`.

## What Is Simulated

- Persistence, parsing, saving, publishing, field
  reordering, and validation are simulated for design review.
- Form controls show intended states but do not write real `.idea` files.

## Safe To Delete

This folder is a review draft and can be deleted after the approved design is
implemented elsewhere.
